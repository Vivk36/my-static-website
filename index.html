📌 About This Project

This website is deployed automatically using a Jenkins CI/CD Pipeline. Every time I push code to GitHub, Jenkins automatically deploys it to this server.

🛠️ Tech Stack Used

Jenkins GitHub Webhooks AWS EC2 Nginx Ubuntu Git

⚙️ How It Works

1. Git Push → GitHub
↓
2. Webhook → Jenkins
↓
3. Jenkins pulls code
↓
4. Deploy to Nginx
↓
5. Website Live! ✅
